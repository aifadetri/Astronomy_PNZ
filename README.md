# 📚 Astronomy_PNZ

Репозиторий с конспектами, теорией и задачами по астрономии для учеников 8-11 классов.  
Материалы сгруппированы по ключевым разделам курса.

## 📂 Структура

- **astrophysic** — основы астрофизики: излучение, звёзды, эволюция  
- **celestial_mechanics** — небесная механика: законы движения, орбиты  
- **celestial_sphere** — небесная сфера: координаты, системы отсчёта  
- **optics** — оптика: телескопы, свет, наблюдения  
- **time_theory** — измерение времени: календари, звёздное время
- **calculators** — калькуляторы, гайды по выбору и использованию

## 🚀 Как пользоваться

1. Выберите нужный раздел  
2. Изучите теорию (конспекты)  
3. Решите предложенные задачи  
4. При необходимости возвращайтесь к теории
